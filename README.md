# MIPS Simulator

A simulator for a subset of the MIPS architecture. The simulator includes:

* Pipelining
* Data Forwarding
* Branching
* Branch prdiction (2-bit predictor)

# Supported operations

* addi
* and
* sub
* sgt
* lw
* sw
* blt
* jr
* jal
